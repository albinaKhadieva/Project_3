Projektname: Klausurprojekt CPP3 einer C++ menügesteuerten Applikation
CPP - OOP Klassen, Vererbung usw.


Version: g++ (MinGW.org GCC-6.3.0-1) 6.3.0

Installation:
Die Batch-Datei "start.bat" in dem Ordner starten

danach das Programm starten:
Bei Main, eine der Optionen waehlen:


  *******************************************
  * Waehlen Sie bitte eine der Optionen aus *
  * ----------------------------------------*
  * 1. Als Student anmelden					*
  * 2. Als Student einloggen				*
  * 3. Als Administrator einloggen			*
  * 0. Exit									*
  *											*
  * Eingabe Option: 1, 2, 3, 0				*
  *******************************************


Um als Admin sich anzuloggen, sollten Sie die Zugangsdaten benutzen:

Login: 		admin
Passwort: 	geheim

Zugangsdaten eines Students:
Login und Passwort eines der Studenten, die zur Liste von Studenten hinzugefügt wurden.
Oder lege selbst einen Nutzer ein.
***************************************************************************************
